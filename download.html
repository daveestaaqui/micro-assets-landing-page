How to Install

🌐 Chrome, Edge, Brave, Opera, Vivaldi, Arc

  1. Unzip the Chrome pack anywhere on your computer.
  2. Open your browser's extensions page (see URLs below) and turn on Developer Mode.
  3. Click "Load unpacked" and select any extension folder.

Repeat step 3 for each extension you want. Install only what you need.

🦊 Firefox

  1. Unzip the Firefox pack anywhere on your computer.
  2. Open Firefox and go to about:debugging#/runtime/this-firefox
  3. Click "Load Temporary Add-on..." and select the manifest.json file inside any extension folder.
