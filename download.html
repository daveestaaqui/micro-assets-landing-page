How to Install (3 steps)

  1. Unzip the downloaded file anywhere on your computer.
  2. Open your browser and go to chrome://extensions — then turn on Developer Mode (top right).
  3. Click "Load unpacked" and select any extension folder from the unzipped directory.

Repeat step 3 for each extension you want. Install only what you need.

Extension Page URLs by Browser

Android Users

Standard Chrome on Android doesn't support extensions. To use SporlyWorks on mobile, install Kiwi Browser from the Play Store. Then open chrome://extensions in Kiwi, enable Developer Mode, and load the extensions.
